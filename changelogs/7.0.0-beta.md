### English
- Major refactor and cleanup of the codebase
- Hunger damage system now using custom `DamageType` e.g. `"minecraft:starve"`
- Fixed nutrition modifier system and exhaustion modifier system not work
- Fix issue that cause Fabric Loader report the mod's version mismatch
- Added extensive language file support
- Added new config file with info and translations

- Removed legacy config and registry classes, and embedded the config-manager dependency
- Introduced a new code generation system with Java templates and helpers
- Increased default JVM memory, updated mod version and dependencies, and removed obsolete Forge/NeoForge resources

- 对代码库进行了重大重构和清理
